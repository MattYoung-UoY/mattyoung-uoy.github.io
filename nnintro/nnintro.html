Introduction to Neural Networks


In this project I am following the "Make your own Neural Network" book by Tariq Rashid.
The book uses the Python programming language, however I shall be using the Java programming language as it is the one I am most familiar with, and it is the one that I would most likely be using in future projects using Neural Networks.

The code for this project can be found on my github: Neural Networks Introduction.

For the first part of the project, I will be using randomly generate datasets that I have created through my code.
Later on in the project, I will be working on a larger Neural Network to recognise hand-written digits. To train and test this Neural Network I will be using the MNIST Dataset.

Compiler Implementation in Java


For this project I am following along with the Modern Compiler Implementation in Java book by Andrew Appel. A link to a pdf of the book can be found here.

The code for this project can be found on my github: Compiler Implementation in Java.

Tools:


The book recommends to use the JLex Lexer Generator and CUP (Construction of Useful Parsers) Parser Generator tools. However I have decided that instead I would use JFlex as my Lexer Generator.

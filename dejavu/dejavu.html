DeJaVu - Java 2D Game Engine


In this project I am creating a 2D game engine called DeJaVu. The name is from the theme of the game jam in which development of this game engine started.

The GitHub repository for the project can be found here.

Resources:


The DeJaVu engine is developed using the Leightweight Java Game Library (LWJGL) version 2 Java binding for OpenGL and OpenAL. I am also using Slick-Util for image loading.

I am using a tutorial series on 3D Java Game Programming using LWJGL by ThinMatrix for reference. I am also using the Eighth Edition of OpenGL Programming Guide, and the Second Edition of OpenGL 4 Shading Language Cookbook for reference.

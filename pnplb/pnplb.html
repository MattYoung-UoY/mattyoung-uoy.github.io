Plug-&-Play Logic Board


The aim of this project is to design and implement a board that can be easily sourced, and constructed by the general public. The purpose of the board is to provide a physical tool which can be used to help children and young teens explore the basic concepts of digital logic, and some useful digital constructs that can be made using these basic logic gates. A physical board that can be constructed was chosen to implement this concept, rather than a software solution, in an attempt to appeal to children, and to inspire them to look further into digital logic and circuitry.

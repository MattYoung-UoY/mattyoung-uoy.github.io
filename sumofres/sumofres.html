Implementing an LSM on an FPGA


The original aim of the project was to implement a Liquid State Machine(LSM) on a Field-Programmable Gate Array(FPGA). The system would then be benchmarked using NARMA-10 and a final speech recognition task performed with it.

The system was designed and tested using VHSIC Hardware Description Language(VHDL), the GitHub Repository for which can be found here.

A Transpiler was also concieved to convert from a simple textual description of the desired LSM, and produce the necessary VHDL to implement the LSM. The GitHub Repository for this can be found here.
